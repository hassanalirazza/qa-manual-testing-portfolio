# Test Case Template

| Field | Description |
|---|---|
| **Test Case ID** | Unique identifier (e.g., TC_LOGIN_001) |
| **Title** | Short summary of what is being tested |
| **Module** | Feature area under test |
| **Priority** | High / Medium / Low |
| **Preconditions** | State required before execution |
| **Test Data** | Inputs needed |
| **Test Steps** | Numbered, repeatable actions |
| **Expected Result** | What should happen |
| **Actual Result** | Filled during execution |
| **Status** | Pass / Fail / Blocked |
